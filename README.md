# Medical RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built using Flask, LangChain, FAISS, Hugging Face Embeddings, and Llama 3.2.

## Features

* PDF-based medical question answering
* FAISS vector database
* LangChain retrieval
* Hugging Face embeddings
* Llama 3.2 answer generation
* Source page citations
* Flask web interface

## Tech Stack

* Python
* Flask
* LangChain
* FAISS
* Hugging Face Transformers
* Sentence Transformers

## Example

Question:
How is HCV transmitted?

Answer:
HCV is most efficiently transmitted by infected blood-to-blood contact.
